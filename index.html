About

Teduh Unit Counter is a Chrome extension that helps you quickly count sold and unsold units on teduh.kpkt.gov.my project pages.

  • See total, sold, and unsold units instantly
  • Save and revisit project pages easily
  • Generate direct links using APDL code
Download Extension

How to Use

  1. Download and install the extension from the link above
  2. Open a project page on teduh.kpkt.gov.my
  3. Click the extension icon to view unit counts and project info
  4. Use the dropdown to revisit saved projects
  5. Generate direct links using APDL code if needed
